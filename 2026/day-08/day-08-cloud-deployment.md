# Day 08 - Cloud Server Setup: Docker, Nginx & Web Deployment

## Objective

Deploy a real web server on a cloud instance and learn practical server administration tasks.

## Tasks Completed

### Cloud Instance Setup

* Launched an Ubuntu cloud server
* Connected securely using SSH
* Verified server connectivity and operating system details

### Nginx Installation

* Updated package repositories
* Installed Nginx web server
* Started and verified Nginx service status

### Security Configuration

* Configured Security Group rules
* Allowed HTTP traffic on port 80
* Verified public access from a web browser

### Log Management

* Viewed Nginx access logs
* Exported logs to nginx-logs.txt
* Prepared logs for download and troubleshooting

## Key Learning Outcomes

* Cloud server provisioning
* SSH-based server management
* Nginx installation and configuration
* Security Group management
* Web service verification
* Log collection and analysis

## Why This Matters

These are fundamental DevOps skills used daily in production environments. Every web application requires secure server access, web server configuration, networking setup, and log monitoring.
